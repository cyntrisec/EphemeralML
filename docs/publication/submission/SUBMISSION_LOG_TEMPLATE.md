## Post-Submission Evidence Capture

After each submission, complete these steps:

1. [ ] Record confirmation number/URL above
2. [ ] Save screenshot of confirmation page to `docs/publication/submission/screenshots/`
3. [ ] Update `startup-plans/STATE.yaml` (A-011 for IETF, A-013 for NIST)
4. [ ] Git commit the updated log: `git add docs/publication/submission/SUBMISSION_LOG_TEMPLATE.md && git commit -m "docs: record [IETF/NIST] submission"`
5. [ ] Push to origin/main
